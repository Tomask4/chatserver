SERVER SETUP GUIDE

Host your own chat server

Kanaal works with a simple Node.js WebSocket server that you host yourself. Follow the steps below — no prior server experience needed.

Requirements

✓
Node.js ≥ 18 — Download at nodejs.org. Verify with node --version
✓
A server or VPS — Start on your own machine (localhost), or host on Railway, Render, DigitalOcean, etc.

Installation

1
Clone the repository
git clone https://github.com/kanaal-chat/server
cd server
2
Install dependencies
npm install
3
Start the server (default port 3001)
npm start
# Or with a custom port:
PORT=8080 npm start
4
Connect! Enter in the connection screen: ws://your-ip:3001 — or wss:// if you're using HTTPS.

Run as a background service (pm2)

npm install -g pm2
pm2 start server.js --name kanaal
pm2 save && pm2 startup

Full server code (server.js)

Want to build the server yourself instead of cloning? Here's the complete code:

// server.js — Kanaal Chat Server // npm install ws const WebSocket = require('ws'); const PORT = process.env.PORT || 3001; const wss = new WebSocket.Server({ port: PORT }); const clients = new Map(); // ws → { name, color } const COLORS = ['#e8ff47','#47ffe8','#ff47e8','#ff8a47','#47a8ff']; function broadcast(msg, except = null) { const data = JSON.stringify(msg); for (const [ws] of clients) if (ws !== except && ws.readyState === WebSocket.OPEN) ws.send(data); } function userList() { return [...clients.values()].map(u => ({ name: u.name, color: u.color })); } wss.on('connection', (ws) => { let user = null; ws.on('message', (raw) => { let msg; try { msg = JSON.parse(raw); } catch { return; } if (msg.type === 'join') { const name = String(msg.name || 'Anon').trim().slice(0, 24); const color = COLORS[clients.size % COLORS.length]; user = { name, color }; clients.set(ws, user); ws.send(JSON.stringify({ type: 'welcome', name, color, users: userList() })); broadcast({ type: 'system', text: `${name} joined the chat` }, ws); broadcast({ type: 'users', users: userList() }); } else if (msg.type === 'message' && user) { const text = String(msg.text || '').trim().slice(0, 2000); if (!text) return; broadcast({ type: 'message', name: user.name, color: user.color, text, time: Date.now() }); } else if (msg.type === 'typing' && user) { broadcast({ type: 'typing', name: user.name }, ws); } }); ws.on('close', () => { if (user) { clients.delete(ws); broadcast({ type: 'system', text: `${user.name} left the chat` }); broadcast({ type: 'users', users: userList() }); } }); }); console.log(`Kanaal server running on port ${PORT}`);

Environment variables

PORT=3001  # Port to listen on (default 3001)

For HTTPS/WSS you need a reverse proxy (nginx, Caddy) or use a platform like Railway or Render — they handle SSL automatically.
